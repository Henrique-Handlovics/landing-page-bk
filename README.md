Landing Page BK 🍔

Projeto de landing page responsiva desenvolvido com HTML, CSS e JavaScript, focado em praticar estruturação de layout, organização de código e estilização de páginas web.

<hr>

📌 Sobre o projeto

Esta aplicação simula uma landing page promocional inspirada em uma marca de fast food, com objetivo de treinar conceitos de front-end e construção de interfaces modernas para web.

O projeto foi desenvolvido como prática de estudos em desenvolvimento web.

<hr>

🚀 Tecnologias utilizadas

HTML5

CSS3

JavaScript

<hr>

🎯 Funcionalidades

Estrutura de página moderna

Layout organizado em seções

Estilização personalizada

Interface simples e visual atrativo
